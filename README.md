## Sunnyside Landing Page
A fully responsive landing page built as a Frontend Mentor challenge, using only HTML and CSS. The project focuses on layout accuracy, modern CSS techniques, and clean code structure.

## Overview
This project is a recreation of the Sunnyside Agency Landing Page from Frontend Mentor. It includes responsive layouts for mobile and desktop, along with well-structured and semantic HTML.

## Features
Fully responsive for mobile, tablet, and desktop
Clean and semantic HTML markup
Modern CSS (Flexbox, Grid, custom properties)
Pixel-perfect design matching the challenge
Organized and scalable file structure

## Tech Stack
HTML5
CSS3 (Flexbox, Grid, Variables)
Frontend Mentor design files
